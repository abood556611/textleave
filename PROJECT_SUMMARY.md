# TextLeaf - Project Summary

## نظرة عامة

TextLeaf هو تطبيق ويب (SaaS) متخصص في توليد فيديوهات "Match Cut" احترافية تحاكي تقليب صفحات المجلات بسرعة. يوفر التطبيق تجربة بصرية واقعية وغير مملة مع تنوع في الخلفيات والنصوص المحيطة.

## المنطق الرياضي

### حسابات الفيديو
```
مدة الصفحة = 0.5 ثانية
عدد الصفحات = المدة الكلية ÷ 0.5

مثال:
- فيديو 10 ثواني = 20 صفحة
- فيديو 15 ثانية = 30 صفحة
```

### تركيبة كل صفحة
1. **النص الرئيسي**
   - الموقع: مركزي مع تنويع عشوائي (±5px)
   - الحجم: 80pt مع تنويع (±2%)
   - اللون: أسود (#000000)

2. **الخلفية**
   - اختيار عشوائي من مكتبة الخلفيات
   - تنوع في الألوان (بيج، كريم، أبيض زهري)
   - تأثير الورق المجعد

3. **النصوص المحيطة**
   - نصوص Lorem Ipsum عشوائية
   - حجم الخط: 30pt
   - ضبابية خفيفة (Blur)
   - شفافية: 100/255

4. **مستطيل التحديد**
   - لون أصفر شفاف (255, 255, 153, 200)
   - حواف مهتزة (Shaky Edges)
   - حشوة: 20px

- **Framework**: Flask 3.0.0

- **Video Processing**: MoviePy 1.0.3
- **Server**: Gunicorn 21.2.0

- **HTML5** مع دعم RTL
- **CSS3** مع Dark Mode
- **Vanilla JavaScript**
- **Font Awesome** للأيقونات

### Database

- جداول: users, subscriptions, videos

### Payment
- **OxaPay** للعملات المشفرة
- **Supported**: USDT (BSC, TRX, ETH)

### Deployment
- **Server**: Nginx
- **Process Manager**: Systemd
- **SSL**: Certbot (Let's Encrypt)
